# Qt Blackjack

**Description:**  
Simple Blackjack game implemented in Qt 6 (Widgets), using C++ with proper class separation.  
Demonstrates use of:

- Classes (`Card`, `Deck`, `Hand`, `Game`)  
- Signals and slots in Qt  
- Clean UI separation  
- Smart use of Qt containers  

**How to build:**  
```bash
mkdir build
cd build
cmake ..
cmake --build .
